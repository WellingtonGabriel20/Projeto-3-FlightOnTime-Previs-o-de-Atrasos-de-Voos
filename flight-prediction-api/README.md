
# Flight on time ✈️

A REST API for flights that predicts whether a flight will be delayed or on time.

## Endpoints
![endpoints](src/main/resources/static/img/endpoint.png)

## Functionalities
- Predict - Tells if a flight is gonna be on time or delayed.

## Requirements
- Java 21 or +;
- Maven;
- An IDE that supports Java;

## How to use:
- Open the project in your favorite IDE;
- Run the FlightPredictionApiApplication class;
- Once running, you can access the project SpringDoc documentation [here](http://localhost:8080/swagger-ui/index.html#/) and test the endpoint;

## Technologies
- Backend: Java 21, Spring Boot 4
- Documentation: SpringDoc
- Versioning: Git, GitHub
